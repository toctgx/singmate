# Sigmate Wireframe v1

## 서비스 한 줄 정의
코인노래방 함께 갈 친구를 위치 기반으로 찾고, 15초 미리듣기 음성 프로필로 매칭되는 앱.

## 핵심 기능
- 15초 미리듣기 등록
  - 직접 녹음
  - 파일 업로드
  - 15초 초과 시 0~15초 자동 잘라서 등록
- 싱메이트 신청
- 싱배틀 신청
- 위치 기반 주변 코인노래방 찾기
- 채팅
