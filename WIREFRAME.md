# Singmate Final Screen Structure v1

## 전체 앱 구조
하단 네비게이션은 총 4개입니다.
1. 홈
2. 채팅
3. 주변 코인노래방
4. 마이페이지

로그인/회원가입은 하단 네비게이션에 포함되지 않는 초기 진입 플로우입니다.

---

## 0. 로그인 / 회원가입 화면
- 로그인과 회원가입을 한 화면 안에서 전환
- 소셜 로그인 + ID/PW 로그인 지원
- 회원가입 시 단계형 입력

### 회원가입 입력 항목
- 휴대폰 번호
- 인증번호
- 비밀번호
- 닉네임
- 성별
- 나이
- 지역
- 프로필 사진
- MBTI
- 좋아하는 노래 장르
- 15초 미리듣기 등록
  - 직접 녹음
  - 파일 업로드
  - 15초 초과 시 앞 15초 자동 등록

---

## 1. 홈 화면
Singmate의 가장 중요한 메인 화면입니다.

### 핵심 구조
- 스와이프 카드 구조
- 카드 안에서 바로 프로필 확인
- 15초 미리듣기
- 싱메이트 신청
- 싱배틀 신청

### 카드에 보이는 정보
- 프로필 사진
- 이름
- 나이
- 지역
- MBTI
- 좋아하는 노래 장르
- 15초 미리듣기 버튼
- 싱메이트 버튼
- 싱배틀 버튼

### 추가 확정 기능: 15초 미리듣기 보상
- 다른 사람 프로필의 15초 미리듣기를 끝까지 들으면 싱코인 1개 지급
- 하루 최대 3개까지만 지급
- 3개 이후에도 미리듣기는 계속 가능
- 단 코인만 추가 지급 안 됨

---

## 2. 채팅 화면
### 역할
- 매칭 후 대화
- 시간 조율
- 코인노래방 장소 조율

### 구조
- 카카오톡처럼 익숙한 채팅 UI
- 상단에 상대 닉네임/상태 표시
- 하단 메시지 입력창

### 싱배틀 추가 기능
싱배틀이 성사되면, 배틀 종료 후 결과 선택 화면으로 연결됩니다.

#### 배틀 결과 입력
- 내가 이겼어요
- 상대가 이겼어요
- 무승부 (선택 가능 시)

이 결과는 누적되어 총 승리 수, 승률, 우리동네 최다 위너 같은 랭킹으로 확장됩니다.

---

## 3. 주변 코인노래방 화면
### 역할
- 내 주변 코인노래방 찾기
- 실제 약속 장소 선택

### 구조
- 현재 위치 기반 리스트/지도
- 매장명
- 거리
- 이동 시간
- 상세보기

### 추가 확정 기능: 위치 공유
- 특정 코인노래방을 선택하면 채팅창에 바로 공유 가능
- 채팅과 직접 연결되는 구조

---

## 4. 마이페이지
### 핵심 역할
- 내 프로필 관리
- 15초 미리듣기 관리
- 신청/매칭 관리
- 싱코인 관리
- 배틀 기록 확인

### 구성
#### 프로필
- 프로필 사진
- 닉네임
- 나이
- 지역
- MBTI
- 좋아하는 장르
- 프로필 수정

#### 15초 미리듣기 관리
- 재생
- 다시 녹음
- 파일 재업로드

#### 신청 / 매칭 관리
- 보낸 싱메이트 신청
- 받은 싱메이트 신청
- 보낸 싱배틀 신청
- 받은 싱배틀 신청
- 매칭 완료 내역

#### 싱코인
- 보유 코인
- 충전하기
- 사용내역

#### 배틀 기록
- 총 배틀 수
- 총 승리 수
- 승률
- 우리동네 랭킹

#### 설정
- 알림 설정
- 계정 관리
- 로그아웃
- 회원탈퇴

---

## 핵심 서비스 정의
이 앱의 본질은 다음 세 가지입니다.
1. 15초 미리듣기 음성 프로필
2. 스와이프 기반 노래 친구 매칭
3. 싱메이트 / 싱배틀 신청과 결과 기록

즉 단순 소개팅앱이 아니라, 노래 취향과 음성 샘플 기반의 코인노래방 매칭 앱입니다.

---

## 메인 플로우
로그인/회원가입
→ 프로필 생성
→ 15초 미리듣기 등록
→ 홈 스와이프 카드
→ 싱메이트 / 싱배틀 신청
→ 채팅
→ 주변 코인노래방 공유/조율
→ 마이페이지에서 기록/코인 관리
