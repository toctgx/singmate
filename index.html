0. 로그인 / 회원가입

  • 한 화면 안에서 로그인 / 회원가입 전환
  • 소셜 로그인 + ID/PW 로그인
  • 휴대폰 번호, 인증번호, 비밀번호, 닉네임
  • 성별, 나이, 지역, 프로필 사진, MBTI, 좋아하는 노래 장르
  • 15초 미리듣기 등록 포함

1. 홈 화면

Singmate의 핵심 메인 화면입니다. 스와이프 카드 구조 안에 15초 미리듣기와 싱메이트/싱배틀 신청이 바로 들어갑니다.

프로필 사진
이름 / 나이 / 지역
MBTI / 좋아하는 장르
[▶ 15초 미리듣기]
[싱메이트] [싱배틀]

추가 기능

  • 다른 사람 프로필의 15초 미리듣기를 끝까지 들으면 싱코인 1개 지급
  • 하루 최대 3개까지만 지급
  • 3개 이후에도 미리듣기는 계속 가능

2. 채팅

  • 매칭 후 대화
  • 시간 조율
  • 코인노래방 장소 조율
  • 카카오톡처럼 익숙한 채팅 UI

싱배틀 결과 입력

  • 내가 이겼어요
  • 상대가 이겼어요
  • 무승부

3. 주변 코인노래방

  • 현재 위치 기반 리스트/지도
  • 매장명 / 거리 / 이동 시간 / 상세보기
  • 특정 코인노래방을 채팅창에 바로 공유 가능

4. 마이페이지

  • 내 프로필 관리
  • 15초 미리듣기 관리
  • 신청/매칭 관리
  • 싱코인 관리
  • 배틀 기록 확인
  • 설정

메인 플로우

로그인/회원가입 → 프로필 생성 → 15초 미리듣기 등록 → 홈 스와이프 카드 → 싱메이트/싱배틀 신청 → 채팅 → 주변 코인노래방 공유/조율 → 마이페이지에서 기록/코인 관리
